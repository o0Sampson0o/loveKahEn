Background music playlist.

The page plays these files in random order (never the same song twice in
a row), with a 3-second gap between songs. Current playlist (see PLAYLIST
in scripts/main.js):

  song.mp3
  song1.mp3
  song3.mp3

To add/remove/rename songs: drop the .mp3 here and update the PLAYLIST
array in scripts/main.js to match the filenames.
